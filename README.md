# UTS ASJ - Microservice Docker

Project ini adalah implementasi microservice menggunakan Docker Compose.

## Services
- Frontend : Port 3000
- Backend : Port 8081
- API : Port 8080
- PostgreSQL : Port 5432
- MinIO : Port 9001

## Cara Menjalankan

Clone repository

git clone https://github.com/sheikha-13/uts-asj.git

Masuk folder project

cd uts-asj

Jalankan docker compose

docker compose up -d
